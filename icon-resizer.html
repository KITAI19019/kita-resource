🛠️ ICO 缩放调节器 — 让你的任务栏图标整整齐齐

KITA | 2026年04月28日 | 工具

起因

事情是这样的——我一直喜欢自定义 Windows 任务栏图标。像 Chrome、VSCode 这些软件的图标都设计得很好看，大小也很统一。但当我想给 GitHub Desktop 换上自己设计的图标时，问题就来了：图标大小总是对不齐。

明明都是 256×256 的 ICO 文件，钉到任务栏上一看，GitHub 的图标就是比 Chrome 小一圈。不管我怎么调整源图片，肉眼看上去就是不和谐。

▲ 调整前：图标大小参差不齐，视觉上很不统一

原因分析

研究之后发现，问题出在图标主体的「视觉面积」上。Chrome 的图标是一个接近正圆的形状，几乎填满了整个画布；而 GitHub 的 Octocat 是一个不规则图形，四周有大量留白。虽然像素尺寸一样，但人眼感知到的大小完全不同。

专业的图标设计中，设计师会通过一个叫「光学尺寸校正」（Optical Sizing）的技巧来解决这个问题——对不同形状的图标施加不同的缩放比例，让它们看起来一样大。但对于普通用户来说，手动用 PS 一个个调太麻烦了。

于是我做了这个工具

ICO 缩放调节器是一个完全运行在浏览器中的轻量级工具，专门解决这个痛点：

  • 一键上传：支持 PNG、JPG、WebP 格式
  • 实时缩放：通过滑块自由调整图标主体大小，去除多余留白
  • 视觉参考：内置"标准参考圆"，助你对齐 Chrome、Gmail 等主流图标的视觉面积
  • 原生导出：直接生成标准的 256×256 .ico 格式，支持透明度
  • 隐私安全：所有图片处理均在本地浏览器 Canvas 完成，不会上传到任何服务器

使用方法

  1. 上传：将你想要修改的图标文件拖入上传区
  2. 缩放：拖动滑块，观察图标主体，使其边缘接近虚线"参考圆"
  3. 下载：点击"保存并下载 ICO"，然后在 Windows 中替换原有的快捷方式图标即可
